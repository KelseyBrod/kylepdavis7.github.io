Flexbox

Explanation

A flexbox can be used to make responsive design. There is a flex container and then elements inside that flex container which contain flex items.

You can change the flex container to wrap the boxes so it will cover multiple lines. Along with that, a flexbox can be formatted to grow or shrink and also change the axis that it flexes on. You can justify the contents so that they are evenly spaced out, pushed to the right or left, or so they all have the same amount of space inbetween them. There are many commands for flexboxes and they can be found here! https://yoksel.github.io/flex-cheatsheet/
